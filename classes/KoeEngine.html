    Class KoeEngine

    Main-thread API for the koe concatenative synthesis engine.

    Loads a single .koe archive. The full voice bank is never held in memory: phonemes are fetched on demand (via VoiceBank) and cached in the audio thread.

    const engine = new KoeEngine(); await engine.load(koeBlobOrUrl); await engine.play([{ phoneme: 'a', pitch: 440, duration: 48000 }]);

    For analysis / resynthesis (e.g. Worldline) you usually only need the raw samples — use VoiceBank directly instead, which needs no AudioContext or worklet.
