    Class VoiceBank

    Read-only access to a .koe voice bank: its manifest plus per-phoneme PCM, fetched on demand (Blob slice or HTTP Range). The full bank is never held in memory.

    Pure data — no AudioContext, no AudioWorklet, no DOM. Use this when you only need the source samples (e.g. to feed the Worldline renderer or any other vocoder). KoeEngine builds its concatenative playback on top of this same class.

    const bank = await VoiceBank.load(koeBlobOrUrl); const pcm = await bank.getPcm('a'); // Float64 [-1, 1]
